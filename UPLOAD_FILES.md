# Upload Files (Public Repo)

Safe to publish:

1. `README.md`
2. `SKILL.md`
3. `scripts/run_pipeline.py`
4. `references/onboarding-template.md`
5. `references/config-schema.json`
6. `references/report-template.md`

Do NOT publish:

- Any `.secrets/*` files
- Any `MEMORY.md`, `USER.md`, daily `memory/*.md`
- Any generated resumes (`*.docx`) or zip packages
- Any personal resume source documents
- Any runtime logs that include private links/tokens
